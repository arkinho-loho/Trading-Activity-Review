# Trading Activity Review (股票交易活动回顾)

[![Language](https://img.shields.io/badge/Language-Python-blue.svg)](https://www.python.org/)

一个基于同花顺交易软件导出的交割单，对股票交易活动进行深度分析和复盘的工具。它可以帮助您计算关键的交易指标，如胜率、赔率和凯利仓位，并生成结构化的数据表格和分析报告。

## ✨ 功能特性

- **自动化分析**: 只需提供同花顺导出的 `xlsx` 格式交割单，即可自动完成所有分析。
- **核心指标计算**: 精确计算胜率、赔率（盈亏比）和凯利仓位，为您提供客观的交易表现评估。
- **FIFO 交易配对**: 采用先进先出 (FIFO) 算法，智能配对买入和卖出记录，构建完整的已平仓交易。
- **多维度统计**:
  - 按**证券类型**分析 (个股, ETF, 可转债)。
  - 按**持股周期**分析 (超短线, 短线, 中线等)。
- **详细报告输出**:
  - 生成 `Excel` 文件，包含所有原始数据、配对过程和指标汇总。
  - 生成 `Markdown` 格式的分析报告，提供综合评估和可视化图表。
- **纯本地运行**: 所有计算均在本地完成，不依赖网络获取实时价格，保证了分析的稳定性和客观性。

## 🚀 如何使用

### 准备工作

1.  从同花顺交易软件中导出您的交割单，并保存为 `.xlsx` 文件。
2.  确保您的环境中已安装 Python。

### 安装依赖

```bash
pip install -r requirements.txt
```

### 运行分析

通过命令行运行 `analysis.py` 脚本，并传入您的交割单文件路径。

```bash
python scripts/analysis.py "路径/到/您的/交割单.xlsx"
```

脚本运行成功后，将在交割单文件所在的目录下创建一个名为 `股票分析_YYYY-MM-DD` 的文件夹，其中包含了分析结果。

## 📊 输出文件说明

### 1. 数据表格 (`.xlsx`)

一个包含多个 Sheet 的 Excel 文件，是所有计算的详细记录。

- **核心指标**: 最终计算出的胜率、赔率、凯利仓位等。
- **持仓明细**: 根据交易记录计算出的当前持仓情况（不包含实时价格）。
- **配对交易**: 所有成功配对的已平仓交易明细。
- **分类统计**: 按证券类型和持股周期的分类汇总统计。

### 2. 分析报告 (`.md`)

一份 Markdown 格式的综合分析报告，内容包括：

- **数据概览**: 交易记录总数、时间范围等。
- **核心指标分析**: 对胜率、赔率、凯利仓位的详细解读。
- **综合评估与建议**: 基于您的交易数据，提供客观的评估和资金管理建议。
- **多维度分析图表**: (未来功能) 提供可视化的图表来展示不同维度下的交易表现。
- **处理日志**: 记录分析过程中的所有警告和错误信息。

## 核心指标解释

- **胜率**: `盈利的交易笔数 / 总交易笔数`
  - 反映了您交易成功的频率。

- **赔率 (盈亏比)**: `平均单笔盈利金额 / 平均单笔亏损金额`
  - 反映了您愿意用多大的亏损去博取一次盈利。一个大于 1 的赔率是比较理想的。

- **凯利仓位 (Kelly Criterion)**: `(胜率 * 赔率 - (1 - 胜率)) / 赔率`
  - 一个用于资金管理的数学公式，旨在找到在保护本金的同时最大化长期收益的最佳仓位比例。计算结果仅供参考。
